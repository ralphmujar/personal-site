Fun with xargs


All of a sudden I wanted to know the size of some binary file I know there is this unix command called du it will give me the size of the file but the problem is I dont know where the file is! luckily I know one command called which that will give me the exact location of the file on the fs so I think finding the file is not really a problem. but I got another prob, how can I combine these commands? It turns out there is a unix command call xarg it standard inputs can be pass as parameter to other commands

> which bash | xargs du -h

Output:

720k /bin/bash
